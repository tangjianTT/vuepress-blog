#### 目录解析：
- `sc-springcloud`：父级目录
   - `sc-api-gateway`：网关服务
   - `sc-common`：工具包
   - `sc-eureka-server`：Eureka注册发现
   - `sc-provide-server`：提供者服务
   - `sc-rabbitmq-server`：RabbitMQ服务
   - `sc-security-auth`：统一登录
